Date: 2026-05-16

Summary: A community member has written a beginner-friendly guide for new Linux and Raspberry Pi users.

A community member has put together [FieldStation42 For Dummies](https://docs.google.com/document/d/1SuhWGHZ0qCLkS23DFLHndt3QxuvK2_LqV-8r33BBL5s/edit?tab=t.0) -- a guide aimed at people who are new to Linux and Raspberry Pi and want to get FieldStation42 up and running without any assumed knowledge.

The official docs are written with some familiarity with the command line in mind. This guide starts earlier than that, covering the basics of navigating Linux so that the rest of the setup makes sense.

If you've been putting off trying FieldStation42 because the terminal felt intimidating, this is the place to start.
