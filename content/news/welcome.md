Summary: FieldStation42 now has a proper home on the web. Here's what's new and what's coming.

FieldStation42 has a new home. After living on the GitHub wiki since the project started, the docs are moving here — to a place that can grow with the project and feel a bit more like the retro TV experience it's all about.

## What's here now

The full getting started guide has been migrated and reworked for clarity, with non-developer users in mind. The Channel Recipes guide is here too, along with the Advanced Scheduling reference.

## What's coming

Feature announcements will live here going forward. If you want to keep up with what's new in FieldStation42, this is the place.

The [YouTube channel](https://youtube.com/@wrongdog_reckons) continues to be the best place for video walkthroughs and deep dives.

## Thank you

FieldStation42 passed 1,300 GitHub stars. That's genuinely wild for a project about pretending your computer is a 1987 cable box. Thanks for being part of it.
